# static_page 
Diseño y estructura Web

*Este proyecto consiste en una landing page desarrollada con HTML5 y CSS3. El objetivo principal fue crear una interfaz informativa, limpia y con una estética moderna inspirada en la cultura del surf.

Tecnologías y conceptos Aplicados
*HTML5: Uso de etiquetas estructurales para un código limpio y estructurado.

*CSS: Implementación de un sistema de diseño basado en variables globales de CSS para mantener la consistencia en la paleta de colores y tipografía, permitiendo la reutilización de código y modularidad del sistema.

*Maquetación con Flexbox: Arquitectura de contenedores flexibles para lograr un diseño alineado y adaptable a diferentes resoluciones.


Organización del Proyecto

*Navegación: Barra superior con posicionamiento sticky para acceso rápido a las secciones.

*Secciones Informativas: Layouts alternados de imagen y texto que utilizan el sistema de cajas de CSS para guiar el ojo del usuario. Además
se incluyeron tarjetas con efecto para que la página no quede tan plana.

*Formulario de Contacto: Diseño con efectos de profundidad mediante sombreados y transparencias. Se aplicó mecanismo para que los campos sean rellenados de manera obligatoria, y se dejó preparado con el "method = post " por si en un futuro se trabaja con bases de datos, poder enviar la información. 

*Footer: Un cierre de página sencillo, con una frase tíipica de surf y mi nombre completo.


Detalles de Diseño (UI)

*Efectos : Micro-interacciones mediante las pseudo-clases :hover y :focus para mejorar el feedback visual en botones e inputs.

*Jerarquía Visual: Uso estratégico de font-weight y line-height para diferenciar contenidos y facilitar la lectura de datos informativos, permitiendo tener una tipografía e interlineado amigable a la visual del usuario.

*Contenedores: Uso de un max-width en las secciones para asegurar que el contenido se vea equilibrado en pantallas de gran formato y que los texto no queden con oraciones tan largas.
