# static_page 
Diseño y estructura Web

*Este proyecto consiste en una landing page desarrollada íntegramente con HTML5 y CSS3. El objetivo principal fue crear una interfaz informativa, limpia y con una estética moderna inspirada en la cultura del surf.

Tecnologías y conceptos Aplicados
*HTML5: Uso de etiquetas estructurales para un código limpio y estructurado.

*CSS: Implementación de un sistema de diseño basado en variables CSS para mantener la consistencia en la paleta de colores y tipografía, declarando variables globales para la reutilización de código y modularidad del sistema.

*Maquetación con Flexbox: Arquitectura de contenedores flexibles para lograr un diseño alineado y adaptable a diferentes resoluciones.


Organización del Proyecto

*Navegación: Barra superior con posicionamiento sticky para acceso rápido a las secciones.

*Secciones Informativas: Layouts alternados de imagen y texto que utilizan el sistema de cajas de CSS para guiar el ojo del usuario.

*Formulario de Contacto: Diseño con efectos de profundidad mediante sombreados y transparencias.

*Footer: Un cierre de página compacto y funcional para información complementaria.


Detalles de Diseño (UI)

*Efectos : Micro-interacciones mediante las pseudo-clases :hover y :focus para mejorar el feedback visual en botones e inputs.

*Jerarquía Visual: Uso estratégico de font-weight y line-height para diferenciar contenidos y facilitar la lectura de datos informativos.

*Contenedores: Uso de un max-width global para asegurar que el contenido se vea equilibrado en pantallas de gran formato.
